Amazon Redshift – Data Warehousing & Analytics

Loaded large, transformed datasets into Redshift using Spark, AWS Glue, and Redshift COPY commands from S3 for efficient batch processing.
Optimized Redshift tables using appropriate distribution keys, sort keys, and column encoding to enhance query performance and reduce scan time.
Built analytical views and reporting tables to support BI dashboards (Power BI, Tableau) and used materialized views to speed up frequent aggregations.

AWS Glue – ETL & Data Cataloging

Developed Glue Spark jobs for batch data transformation, complex joins, and incremental loads using S3 as the storage layer.
Leveraged Glue Crawlers to automatically catalog data from S3 and create table metadata in the AWS Glue Data Catalog for downstream querying in Athena and Redshift Spectrum.

Amazon Athena – Serverless Querying & Data Validation

Used Athena to: Perform ad-hoc SQL queries on S3-stored datasets without provisioning infrastructure.
Validate data quality post-load and reconcile source-to-target record counts.
Analyze log files and processed datasets efficiently using partitioned tables.
Integrated Athena queries within Airflow and Lambda workflows for quick schema validations and reporting.

Amazon CloudWatch – Monitoring & Alerting

Monitored pipeline performance, Lambda executions, and Glue job runs using CloudWatch logs and metrics.
Set up CloudWatch Alarms to trigger SNS notifications and auto-remediation workflows in case of job failures, S3 ingestion issues, or resource thresholds being breached.
Used custom log filters to track specific events like data load counts, transformation errors, and schema mismatches.
